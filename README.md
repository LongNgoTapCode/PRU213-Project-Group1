# PRU213 Coffee Service Game

## Overview
This project is a Unity-based coffee service simulation where players prepare drinks, serve customers under time pressure, manage reputation, and purchase gameplay upgrades.

## Core Gameplay
- Pick up a cup automatically when approaching the cup counter.
- Interact with workstations using `E` when in range.
- Collect ingredients directly into the held cup.
- Serve customers by selecting the correct drink composition.

## Key Systems
- **Order System**: Customer requests and validation flow.
- **Scoring & Economy**: Score for performance, coins for upgrades.
- **Upgrade System**: Brew speed, customer patience, and stability progression.
- **Audio & Menu**: Background music and click/sfx volume controls in Options.

## Project Structure
- `Assets/Scripts/`: Core gameplay and UI scripts.
- `Assets/Scenes/`: Menu and gameplay scenes.
- `Assets/Prefabs/`: Reusable gameplay objects.

## Getting Started
1. Open the project in Unity.
2. Load `BackgroundScene` for menu flow, or `SampleScene` for gameplay testing.
3. Ensure scene references are assigned in Inspector (UI texts, panels, sliders, audio sources).
